## Project Overview

This project analyzes the relationship between offshore wind power generation in Germany and meteorological conditions (wind speed) at key North Sea and Baltic Sea stations.

The goal is to uncover seasonal patterns, correlations between wind conditions and energy output, and long-term trends in offshore wind generation — covering the period from 2021 to 2025, which includes the pre- and post-Ukraine war energy crisis context.

**Target audience:** Energy sector analysts, data teams at companies like Vattenfall, EnBW, or Hamburg Port Authority — roles requiring both technical analysis and clear communication of findings.

---

## Research Questions

- How strongly does daily wind speed correlate with offshore wind power generation?
- Are there clear seasonal patterns in offshore wind production?
- How did the 2022 energy crisis affect offshore wind generation trends?
- Are there differences between North Sea and Baltic Sea wind conditions?
- Is there detectable year-to-year variability in wind conditions during 2021–2025, and does it correlate with production trends?

> **Note on wind variability and climate:** A 5-year window is too short to draw conclusions about climate-driven wind trends — meteorologists require at least 30 years for reliable wind climatology. Year-to-year variability, however, can be meaningfully analysed within this period and provides a foundation for longer-term studies.

## Data Sources

### 1. Offshore Wind Power Generation — SMARD (Bundesnetzagentur)
- **Source:** [smard.de](https://www.smard.de) — official German electricity market data platform
- **Variable:** Wind Offshore realisiert (MWh/day), filter code `1225`
- **Region:** Germany (DE)
- **Period:** 2021-01-01 to 2025-12-30
- **Resolution:** Daily
- **Access:** Via public REST API (`https://www.smard.de/app/chart_data/`)
- **Script:** `scripts/fetch_smard_wind_offshore.py`

### 2. Daily Wind Speed — DWD Open Data (Deutscher Wetterdienst)
- **Source:** [opendata.dwd.de](https://opendata.dwd.de) — German national meteorological service
- **Variable:** Daily mean wind speed in m/s (column `FM` in KL climate files)
- **Period:** 2021-01-01 to 2025-12-31
- **Resolution:** Daily
- **Access:** Direct download of zip files from `historical/` and `recent/` endpoints
- **Script:** `scripts/fetch_dwd_wind_stations.py`

#### Stations selected

| Station | ID | Sea | Rationale |
|---|---|---|---|
| Helgoland | 02564 | North Sea | Island ~70km offshore — best open-sea wind proxy |
| List auf Sylt | 03032 | North Sea | Northernmost exposed tip of Sylt island |
| Fehmarn | 01503 | Baltic Sea | Baltic Sea island — allows North Sea vs. Baltic comparison |

Stations were selected based on their open-sea exposure. Coastal stations sheltered by terrain (e.g. Travemünde, Cuxhaven) were excluded as they do not accurately represent offshore wind conditions.

> **Note on station proximity:** Wind speed measurements are used as a proxy for offshore wind conditions. The DWD stations are the closest available open-sea measurement points, but do not coincide with the actual wind farm locations. German offshore wind farms (e.g. Alpha Ventus, Nordsee One) are located 30–100 km from the coast; Helgoland, at ~70 km offshore, provides the best available approximation.

## Scripts

### `fetch_smard_wind_offshore.py`
Queries the SMARD public API to retrieve daily offshore wind generation data for Germany (2021–2025). The API returns data in time-stamped JSON blocks (Unix timestamps in milliseconds); the script iterates over all available blocks within the target period and exports a clean CSV.

**Key parameters:**
```python
FILTER     = 1225   # Wind Offshore (realisiert)
REGION     = "DE"
RESOLUTION = "day"
```

### `fetch_dwd_wind_stations.py`
Downloads KL (Klimadaten) daily zip files for three stations from both the `historical/` and `recent/` DWD endpoints, extracts daily mean wind speed (`FM`), filters to 2021–2025, and merges all stations into a single CSV.

**Note:** The `historical/` endpoint covers data up to end of 2024; the `recent/` endpoint covers the last ~500 days. Both are needed to cover the full 2021–2025 period.

### `find_dwd_stations.py`
Helper script used during data exploration. Reads `KL_Tageswerte_Beschreibung_Stationen.txt` and filters stations by name keywords (e.g. Helgoland, Sylt, Fehmarn) to identify station IDs and date coverage.

## Analytical Notes

**On the wind speed–production correlation:** The correlation between wind speed and power generation reflects how consistently wind conditions translate into electricity output. However, raw production figures (MWh) are also influenced by installed capacity, which has grown year-on-year as new wind farms came online. A rising production trend does not necessarily mean stronger winds. If data on installed capacity (MW) becomes available, a *capacity factor* analysis (actual output / theoretical maximum) would provide a cleaner measure of generation efficiency.

## Technical Notes

- SMARD API timestamps are Unix milliseconds (UTC). The script applies a small start date offset (`2020-12-25`) to ensure the first available data block covers `2021-01-01`; the extra row (`2020-12-31`) is removed during cleaning.
- DWD missing values are encoded as `-999` and replaced with `NaN` during ingestion.
- All scripts use only standard Python libraries (`requests`, `pandas`) — no API keys required.
